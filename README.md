
# 说明

此仓库用于ZJU的ML_project2，旨在了解神经网络的构建、训练、测试和调参等内容

## 文件

- test_Chess 和 test_MNIST：主文件，包括神经网络的创建、训练、测试

- nn_create:创建一个神经网络的结构体

- nn_train：训练神经网络，包括

（1）nn_forward：前馈

（2）nn_backpropagation：反向传播

- nn_test：测试一个神经网络

## Usage

```bash
python testChess.py
python testMNIST.py
```

## 其他

- 运行代码需要的python包：numpy、matplotlib、scikit-learn
